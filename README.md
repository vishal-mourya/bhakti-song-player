# Bhakti Song Player

## Overview

Bhakti Song Player is a React-based web application designed to provide a seamless and spiritual experience for listening to bhakti (devotional) songs. This player aims to create an immersive environment for users to connect with divine music and enhance their spiritual journey.

## Features

- 🎵 Curated collection of bhakti songs
- 🔄 Shuffle and repeat options
- 📱 Responsive design for various devices
- 🎨 Customizable themes
- 📜 Lyrics display (if available)
- 🔊 Volume control and mute option

## Getting Started

### Prerequisites

- Node.js (v14.0.0 or later)
- npm (v6.0.0 or later)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/vishal-mourya/bhakti-song-player.git
   ```

2. Navigate to the project directory:
   ```
   cd bhakti-song-player
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
